# PASSWORD_LOCKER.

## Author
* Anna Adhiambo

## Description
* An application that will help us manage our passwords and even generate new passwords for us.

## Features
* Create account for the new user
* Create new account
* Display existing account
* Delete account
* Search account by site name
* Exit for leaving the site

## Technology Used
* Python
* Pip
